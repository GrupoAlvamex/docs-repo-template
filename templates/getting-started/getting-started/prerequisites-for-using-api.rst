Prerequisites
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

To start using the API and complete the examples in this section, you need the 
following items:

- Rackspace Cloud account. If you don't have one, `sign up`_.
- :ref:`Command line tool or browser client<send-api-requests>` to communicate with the 
  API service
- Configured environment variables for :ref:`cURL <configure-environment-variables>` 
  command line tool.

.. _Cloud Control Panel: https://mycloud.rackspace.com    
.. _sign up: https://cart.rackspace.com/cloud/
